# ⚙️ Learn C++

Projeto simples para praticar lógica de programação e fundamentos de C++.

## 🌎 Conteúdo estudado

- `if / else`
- `for`
- Funções
- Entrada e saída (`cin` / `cout`)
- Modularização em múltiplos arquivos
- Arrays
- Operações matemáticas
- Tabuada
- Média de valores
- Maior e menor número

## 🏗️ Estrutura do projeto

```bash
learn-cpp/
├── main.cpp
├── arrays.cpp
├── arrays.h
└── CMakeLists.txt
```

# 📚 Exercícios implementados

- Contagem progressiva
- Contagem regressiva
- Números pares
- Números ímpares
- Soma de 1 até 100
- Tabuada
- Média de números
- Maior número
- Menor número

Estudar fundamentos da linguagem C++ e reforçar lógica de programação através de exercícios práticos.
